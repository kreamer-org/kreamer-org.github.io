How to Turn the Keyboard Backlight Off in Lion

Posted on April 14, 2012

OS X Lion has bug that turns on the keyboard backlight when doing Fast User Switching, and then keeps it on after logging in with a user that has disabled the feature. Worse yet, the system doesn't seem to "know" that the backlight is still on. So, you've got to turn it "on" and then back off again to actually turn it off, hitting the associated keyboard key twice. Annoying.

I found out from a co-worker how to get this to stop, and wanted to post here to remember and to help anyone else having the problem. In Terminal, type: 

sudo "/Applications/System Preferences.app/Contents/MacOS/System Preferences"
This runs System Preferences as root, rather than your user. Obviously, you'll need sudo permissions to be able to do this; don't worry, you'll have the correct permissions if you are an admin on the machine. Then, go to the same Keyboard preference pane, and disable "Automatically illuminate keyboard in low light". Reboot your machine, and you should no longer get a backlit keyboard at the login prompt.
